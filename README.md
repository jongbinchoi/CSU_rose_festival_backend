# 조선대학교 장미 축제 



#### **[유입량 페이지]**

+ 전체 총 유입량
+ 당일 유입량
+ 신고된 방명록
  + 신고 수
  + 신고된 방명록 글
  + 신고된  글 삭제
    	+ 신고글 삭제시 마스터 키 입력 후 삭제



#### **[방명록 페이지]**

+ 닉네임 : 랜덤 닉네임(Default) or 사용자 지정 닉네임
+ 글 등록시 비밀번호 설정 <보류>
+ 방명록 내용 : 글자 제한(50자 정도)
+ 신고 기능

---



>  **보류**

+ 방명록 글 편집기능

+ 방명록 글 삭제기능

+ 내가 쓴 글 조회 기능
